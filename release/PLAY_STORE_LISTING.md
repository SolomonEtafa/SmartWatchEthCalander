# Ethiopian Chronograph

## Short description

An analog Wear OS watch face with the current Ethiopian month and date.

## Full description

Ethiopian Chronograph is a clean analog watch face designed for Wear OS.

Features:

- Current Ethiopian month and date in Ge'ez script
- Bold, readable date window
- Analog hour, minute, and second hands
- Optional health or heart-rate complication
- Dark chronograph-inspired design with orange accents

The watch face works directly on the watch and does not require a companion
phone application. Install the separate Ethiopian Calendar Provider watch app
to display the Ethiopian month and date in the date window.
